Tải file xuống, giải nén rồi copy thư mục Dictionary vào ổ D.
Hướng dẫn sử dụng:

*Từ điển Anh-Việt
- Chức năng thêm từ: từ cần thêm là từ tiếng Anh, "Nhập nghĩa" là nghĩa tiếng Việt của từ đã nhập, 
loại từ là từ tiếng Anh (VD: noun, verb, adjective...)
- Chức năng tra từ: từ cần tra là từ tiếng Anh nhập từ bàn phím, 
nếu từ có trong file TuDienA.txt thì sẽ trả lại kết quả 
còn không sẽ hiện thông báo không tìm được từ cần tra.
- Chức năng xóa từ: từ cần xóa là từ tiếng Anh nhập từ bàn phím, 
nếu từ có trong file TuDienA.txt thì từ sẽ bị xóa khỏi file TuDienA.txt 
còn không sẽ hiện thông báo không tìm thấy dữ liệu.
- Nhấn phím Esc để thoát: gõ "y" thì mọi thao tác sẽ được lưu vào file TuDienA.txt,
gõ "n" thì dữ liệu sẽ không lưu vào file TuDienA.txt
